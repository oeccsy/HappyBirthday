# HappyBirthday
> ### 친구를 위한 생일 선물

```
친구를 위한 게임 생일 선물이면서, 제가 처음으로 만들어본 게임입니다.

첫 작품이기 때문에 겉으로 보기에도 서툰 게임이고, 어떻게 보면 짓궃은 선물일 수 있지만
친구는 그 어디에서도 흔하게 받지 못할 선물이라고 말했습니다.

저는 누군가에게 잊지 못할 소중한 경험을 선물하고자 하는 수단으로 게임을 선택했습니다.
```
